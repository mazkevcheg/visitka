# Портфолио и мастер‑генератор портфолио

Небольшой набор статических страниц для личных портфолио и экспериментов с UI:

- `index.html` — одностраничное портфолио‑визитка с тёмной темой, фиолетовыми акцентами, градиентным фоном и поддержкой двух языков (RU/EN).
- `index2.html` — альтернативная версия портфолио с улучшенным оформлением, анимациями и пульсирующим свечением вокруг аватара.
- `portfolio-generator.html` — мастер‑генератор портфолио: задаёт вопросы, показывает живой предпросмотр и помогает собрать структуру личного сайта.
- `portfolio-generator-result-demo.html` — пример вымышленного портфолио, сгенерированного мастером, в том же серо‑чёрном стиле.

## Как запустить

Никаких сборок и зависимостей не нужно — всё на чистом HTML, CSS и JavaScript.

1. Откройте нужный файл в браузере (двойной клик по файлу или перетащите его в окно браузера):
   - `index.html` / `index2.html` — для просмотра статичных портфолио.
   - `portfolio-generator.html` — для работы с мастером (кнопка **«Запустить мастер»** запускает опрос).
   - `portfolio-generator-result-demo.html` — готовый пример результата.
2. Для локального сервера (необязательно) можно использовать любой простой HTTP‑сервер, например:

```bash
cd path/to/project
python -m http.server 8000
```

и открыть `http://localhost:8000` в браузере.

## Как использовать мастер‑генератор

- Откройте `portfolio-generator.html`.
- Нажмите кнопку **«Запустить мастер»** и последовательно заполните шаги:
  имя, роль, блок «О себе», email и ссылки.
- Справа появится сгенерированный макет портфолио.
- Нажмите **«Скачать index.html»**, чтобы сохранить готовый HTML‑файл — его можно открыть в браузере или загрузить на хостинг.

## Кастомизация

- Меняйте тексты прямо в HTML‑файлах — все данные хранятся в разметке или небольших JS‑объектах.
- Цвета и анимации настраиваются через CSS‑переменные и keyframes в `<style>` каждого файла.
